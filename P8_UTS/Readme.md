# Praktikum Pemrograman Web
## Pertemuan 8 (UTS)
Repositori ini berisi proyek Ujian Tengah Semester (UTS) untuk mata kuliah Praktikum Pemrograman Web. Proyek ini diimplementasikan dengan tema Company Profile: IMC Service Center.


## Deskripsi Proyek
Website IMC Service Center adalah platform layanan servis laptop, HP, dan PC yang dirancang untuk memberikan informasi layanan kepada pelanggan. Website ini dikembangkan menggunakan HTML5 dan CSS3 dengan pendekatan multi-page untuk pengalaman navigasi yang intuitif.


## Fitur Utama
- Multi-Page Navigation: Terdiri dari halaman Home, About, Services, Gallery, dan Contact yang saling terhubung.
- Responsive Design: Menggunakan CSS flexbox untuk pengaturan layout yang rapi.
- Modern Interface: Desain Dark Mode dengan aksen warna hijau (#22C55E) untuk kesan profesional.
- Gallery Showcase: Menampilkan dokumentasi layanan dan aktivitas teknisi.
- Semantic HTML: Menggunakan tag (\<header\>, \<nav\>, \<main\>, \<section\>, dan \<footer\>) sesuai standar pemrograman web.

## Teknologi yang Digunakan
- HTML5: Struktur konten web.
- CSS3: Pengaturan gaya, layouting, dan styling responsif.
